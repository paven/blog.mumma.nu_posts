--- layout: post title: getter setters date: '2009-12-16T17:28:00.003+01:00' author: Patrik Gustafsson tags: - java syntax error modified_time: '2009-12-16T17:43:49.488+01:00' blogger_id: tag:blogger.com,1999:blog-4221898218177181968.post-7784726627679278064 blogger_orig_url: http://blog.mumma.nu/2009/12/getter-setters.html --- Jag ogillar allt onödigt lull som hänger med en korekt skriven kod. Men om syntaxen hade varit mer mysig hade det varit mer okej.

Önskar man fick skriva:
private Player owner = null{
public Player getOwner() { return owner; }
public void setOwner(Player owner) { this.owner = owner; }
}
eller fick skriva
private Player owner = null{
public Player get() { return owner; }
public void set(Player owner) { this.owner = owner; }
}
det vore najs... ännu bättre vore om man kunde overrida = för olika relationer.
